## Design Context

### Users
Users are members of the desktop customization ("ricing") community. They use the tool to colorize existing wallpapers to match their system's color schemes (like Catppuccin, Gruvbox, Nord). They want a utility that "just works" and helps them achieve perfect color coordination for their OS.

### Brand Personality
Just works, clean, minimal. The interface should feel functional, straightforward, and unobtrusive.

### Aesthetic Direction
A strictly minimal visual direction. The design should prioritize the tool's core functionality without excessive decoration or visual noise.

### Design Principles
1. **Utility First**: The primary focus is on getting the job done efficiently without distraction.
2. **Clean & Minimal**: Keep the interface uncluttered, stripping away anything that doesn't serve the core purpose.
3. **Unobtrusive Execution**: Avoid unnecessary animations or elements; the tool should feel like it simply "just works."
