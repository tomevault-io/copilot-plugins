# Workout-Tracker

> Tome by [godblessamki](https://github.com/godblessamki/Workout-Tracker) — distributed by [TomeVault](https://tomevault.io)

IronTrack is a native iOS workout tracker built with SwiftUI and SwiftData. It allows users to create custom workout routines and exercises. During active sessions, it automatically displays the maximum weight and reps from the previous session for instant progress tracking. Designed with MVVM architecture, it offers a fast, local-first experience.
